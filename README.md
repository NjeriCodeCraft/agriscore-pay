# 🌾 AgriScore-Pay
**Revolutionizing Agricultural Finance through Data Science & Interswitch**

## 🚀 Overview
AgriScore-Pay is an intelligent credit-scoring engine and marketplace designed to provide farmers with "Buy Now, Pay at Harvest" (BNPH) capabilities. We use Data Science to assess risk and Interswitch to power the payments.

## 🛠️ Technical Stack
- **Frontend:** React / Next.js
- **Backend:** Python (FastAPI) 
- **Data Science:** Scikit-learn / XGBoost (Predictive Yield Modeling)
- **Payments:** Interswitch Webpay & Recurring Tokens API

## 🏗️ Architecture
1. **Risk Assessment:** ML model analyzes historical yield & weather data to set credit limits.
2. **Disbursement:** Interswitch APIs pay input vendors directly.
3. **Recovery:** Automated harvest-time collection via Interswitch Recurring Tokens.
